## Introduction

The Librarian is simple full-stack MERN application with basic CRUD operations. It serves as a virtual library where users are able to input books they've read and basic information about them. Frontend part is made in React (create-react-app), Node and Express serve as backend while data are stored in MongoDB Atlas with Mongoose modeling. It was created for purpose of getting further knowledge & practice of MERN stack, particularly Node and Express.

## Functionality

Users can:
- display books and their details stored in the database
- add book with more information to the database
- modify the information of the book
- delete the book from the database

## Getting started

Inside the project directory install the dependencies & run the server:

### `npm install`
### `npm run app`

Then move to client folder (cd client), install the dependencies & run the app.

### `npm install`
### `npm start`

Launches the app the development mode and opens [http://localhost:3000](http://localhost:3000) automatically.
